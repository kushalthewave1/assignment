What is Full Stack Development?

Full stack development covers both front-end (UI/UX in the browser) and back-end (server, database, APIs), plus tooling for deployment and version control. This portfolio demonstrates core topics with working examples.

Full Stack Roadmap

  • Backend: Node/Express (or other), REST APIs
  • Database: SQL/NoSQL, CRUD operations
  • DevOps: Git, GitHub, Hosting, CI/CD basics
  • Testing: Unit & Integration tests
